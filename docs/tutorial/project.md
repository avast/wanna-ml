# WANNA project
WANNA project settings set some basic values about your project. 

::: wanna.core.models.wanna_project.WannaProjectModel
    :docstring:

### Example

```
wanna_project:
  name: wanna-julia-notebooks
  version: 1
  authors: [harry.potter@avast.com, ronald.weasley@avast.com]
```
